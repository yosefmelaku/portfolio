# My Portfolio

A modern multi-page React portfolio website built with Vite and TypeScript.

## Features

- **Home**: Logo, name, and about me section
- **About**: Detailed personal information
- **Skills**: Technical skills showcase
- **Projects**: Portfolio of completed projects
- **Experience**: Work and internship history
- **CV**: Downloadable CV
- **Contact**: Contact information and links
- **Navigation**: Fixed navigation bar for easy page switching
- **Attractive Backgrounds**: Gradient backgrounds for visual appeal

## Technologies Used

- React 18 with React Router
- TypeScript
- Vite
- ESLint

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) runtime

### Installation

1. Clone the repository or navigate to the project directory
2. Install dependencies:
   ```bash
   bun install
   ```

### Running the Development Server

Start the development server with:
```bash
bun run dev
```

The application will be available at `http://localhost:5173`

### Building for Production

Build the project:
```bash
bun run build
```

Preview the production build:
```bash
bun run preview
```

## Deployment to Vercel

1. Push your code to a Git repository (GitHub, GitLab, etc.)
2. Connect your repository to [Vercel](https://vercel.com)
3. Vercel will automatically detect it as a Vite project and deploy it
4. Your portfolio will be live at a Vercel URL

## Customization

- Replace placeholder content in `src/pages/` with your actual information
- Update the logo in `src/pages/Home.tsx`
- Add your real CV file to `public/cv.pdf`
- Modify colors and styles in the components as needed
